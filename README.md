# mingw-wineggx
* Windows版EGGX を g++ と mingw で開発するためのMakefile

# 必要なもの
* msys2
* mingw-w64-x86_64-toolchain
* make
* Windows 10 SDK
  * すべてはいらない
  * C++ 関連のみでよい

# 使い方
1. 必要なもの に書いてあるものをすべてインストール
1. Makefile の Windows SDK のバージョンを適宜修正
1. make
